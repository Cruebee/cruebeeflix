Documentation

This application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of ALL movies /movies GET None A JSON object holding data about all movies with the following format:

{ 'id': '9', 'title': 'The Boondock Saints', 'author': 'Troy Duffy', 'screenwriter': 'Troy Duffy', 'year': '1999', 'description': '', 'genre': 'Action', 'director': 'Troy Duffy' }

Get data about a single movie by its title /movies/[title] GET None A JSON object containing information about a single movie, including the title, author, screenwriter, year, description, genre, and director.

Example: { 'id': '9', 'title': 'The Boondock Saints', 'author': 'Troy Duffy', 'screenwriter': 'Troy Duffy', 'year': '1999', 'description': '', 'genre': 'Action', 'director': 'Troy Duffy' }

Get data about ALL genres /genres GET None A list of JSON objects holding genre names, and a description about the genre.

{ 'name': 'Comedy', 'description': 'Comedy films are associated with the type of work that is amusing and satirical. Comedies are mostly having a cheerful ending. Tragedy is the opposite to a comedy, as tragedy deals with sorrowful and tragic events in a story.' }

Get data about a specific genre by name /genres/:name GET None A JSON object holding a genre name and a description about the genre same format as loading the genre list
Get data about a director by name /directors/:name GET None A JSON object holding information about a director in the following format:

Example: { 'name': 'George Lucas', 'bio': '', 'age': '' }

Add new user /users POST A JSON object holding data about the user being added, with the following format:

{ 'name': 'user name here', 'username': 'usernameGoesHere', 'password': 'userpassword123', 'email': 'example@email.com', 'birthday': '01/02/1003' }

A JSON object holding the data about the new user, including an ID:

{ 'id': '#', 'name': 'user name', 'username': 'user username', 'password': 'userPassword123', 'email': 'emailexample@email.com', 'birthday': '01/02/1003', 'favorites': [] }

Allow users to update their information with their ID /users/[id] PUT A JSON object holding the data to be updated with the format:

{ 'id': '#', 'name': 'updated name', 'username': 'updated username', 'email': 'updated email', 'birthday': 'updated birthday' }

A JSON object containing the updated information.
Allow users to add movies to a list of favorites using their ID to add a movie by title /users/[id]/[title] POST None A message indicating the movie has been added to favorites
Allow users to remove movies from favorites list /users/[id]/[title] DELETE A message indicating the movie was removed from favorites.
Allow users to delete their account by ID /users/[id] DELETE A message indicating the user account has been removed
